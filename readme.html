สำหรับคนทำเวิร์ดเพรสและชอบใช้ Bootstrap วันนี้เอาไอ้นี่มาแนะนำครับ . UnderStrap . มันคือการจับเอา Underscores ซึ่งเป็น Starter Theme ของทางเวิร์ดเพรสเอง มารวมกับ Bootstrap ที่เป็น CSS Framework ชื่อดัง (ซึ่งตอนนี้ก็เอา Bootstrap 4 เข้ามาใช้แล้ว) โดยธีมตัวนี้ก็รองรับฟีเจอร์อย่างครบถ้วน เช่น - รองรับ Fontawesome - ใช้ SASS - รองรับการทำงานกับ woo commerce - รองรับ JetPack - มี Pagination ให้ในตัว (ไม่ต้องลง WP PageNavi) . และปัญหาโลกแตกอย่างคลาสเมนูที่ฟังก์ชันของเวิร์ดเพรสใส่คลาสไม่ตรงกับ Bootstrap ก็ถูกแก้มาให้เรียบร้อย ด้วยการแถม WP_Bootstrap_Navwalker ซึ่งเป็น Nav Walker มาให้ในตัว . ที่สำคัญคือฟรี และยังพัฒนาอยู่ . ใครทำเวิร์ดเพรส + Bootstrap อยู่แล้ว เชิญโหลดไปลองครับ https://understrap.com/
